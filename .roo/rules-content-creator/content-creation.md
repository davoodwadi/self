# How to create the content

- In each week, you create a content.md file. You should follow the template content-template.md
- You put all the content to discuss during the lecture inside content.md inside that week's directory. You follow the writing.md for the writing style.
- Inside the content space each talking point with questions that spark conversations in class
- Avoid generic questions and make them interesting.
- Next, based on the content inside content.md create the page for the slides. 
- Use all of the points inside the instructions to create a cinematic slide deck.
- Create a 3D background using THREE.js based on what the content.md says.
